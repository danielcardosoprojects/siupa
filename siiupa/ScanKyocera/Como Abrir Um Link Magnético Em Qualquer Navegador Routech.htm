Como abrir um link magntico em qualquer navegador

Internet / novembro 12, 2020 novembro 15, 2020
Reading Time: 4 minutes

Por muito tempo, os links magnticos tm sido o esteio de todos os principais navegadores da web, tornando a vida um pouco mais fcil para as pessoas que baixam arquivos torrent. Mas o que exatamente so elos magnticos? Como voc os configura e abre? E como voc altera o programa ao qual eles se vinculam? Revelamos tudo aqui.

Nota: Voc precisar ter uma plataforma Bittorrent como Vuze, Utorrent ou Transmisso para usar links magnticos.

O que um link magntico?

Primeiro, uma rpida introduo. Links magnticos e arquivos torrent andam de mos dadas, mas no so exatamente a mesma coisa. Na verdade, se voc usar um link magntico, nem precisar baixar um arquivo torrent para comear a baixar o contedo. Suas informaes de identificao cruciais o hash so calculadas em um determinado site, e no no cliente de torrent, e enviadas diretamente para o programa Bittorrent.

Os benefcios disso so que um site no precisa mais hospedar o arquivo torrent original, basta fornecer o link. Para o usurio, isso elimina a necessidade de baixar um arquivo torrent antes de baixar o contedo e, para o site, economiza largura de banda porque no precisa hospedar arquivos em seus servidores.

A grande maioria dos sites de torrent atualmente usa links magnticos.

Abrir links magnticos no Chrome

Primeiro, voc precisa se certificar de que o Chrome permitir que voc crie links magnticos. Para fazer isso, v para Configuraes, clique no cone do menu no canto superior esquerdo, Avanado e em Privacidade e segurana.

Em seguida, clique em Configuraes de contedo -> Manipuladores e Permitir que os sites peam para se tornarem manipuladores padro para protocolos (recomendado).

Voc s precisa encontrar um site de download de torrent que hospede links magnticos (a maioria deles faz), selecionar o arquivo que deseja baixar e escolher Download magntico, se houver tal opo. Caso contrrio, basta selecionar a opo de download padro e h uma boa chance de que seja um link magntico.

Se for um link magntico que voc est baixando, voc receber uma mensagem oferecendo a abertura do link magntico com a plataforma Bittorrent de sua escolha.

No nosso caso, o Vuze. Quando o Chrome perguntar se voc deseja abrir seu cliente Bittorrent, marque a caixa que diz Sempre abra esses tipos de links no aplicativo associado.

Abra os links magnticos no Firefox

Para abrir links magnticos no Firefox, voc encontra novamente um arquivo torrent que deseja baixar e um pop-up deve aparecer perguntando com qual programa (geralmente o seu cliente de bit torrent) deseja abrir o arquivo. Para redefinir a associao e fazer com que o dilogo aparea novamente, faa o seguinte.

Na barra de endereo do Firefox, digite about:config, ento, nessa pgina, pesquise por network.protocol-handler.expose.magnet. Clique com o boto direito nesta preferncia e clique em Alternar para se certificar de que o Valor est definido como falso.

Abrir links magnticos em outros navegadores

Se voc j tem um cliente Bittorrent, o processo de associao de links magnticos em outros navegadores bastante semelhante. Basta encontrar o arquivo que deseja baixar, clicar nele e, se for um link magntico, seu cliente Torrent dever abrir automaticamente ou voc poder receber uma mensagem perguntando se deseja abrir o arquivo magntico com seu aplicativo Bittorrent, como esta mensagem no Microsoft Edge ilustrado abaixo.

Se nada acontecer quando voc clicar no link magntico ou receber a mensagem este arquivo no tem um aplicativo associado a ele, ser necessrio criar a associao do arquivo manualmente.

V para o Editor de registro do Windows (Ganhar + R, entrar regedit na caixa) e navegue at o seguinte local:

ComputerHKEY_CLASSES_ROOTMagnetshellopencommand

No painel de registro direita em Dados, ele deve mostrar o diretrio do seu aplicativo Bittorrent seguido por algum outro texto. Portanto, para Utorrent em seu diretrio padro, deve ser assim:

"C:UsersUserAppDataRoaminguTorrentuTorrent.exe" "%1" /SHELLASSOC

Se as informaes em Dados no corresponderem ao diretrio do cliente Bittorrent, clique com o boto direito em (Padro), clique em Modificar e digite o diretrio correto para o executvel do cliente Bittorrent, certificando-se de que ele seja seguido por "%1" /SHELLASSOC.

Depois de fazer isso, todos os seus navegadores devem associar links magnticos ao aplicativo Bittorrent de sua escolha.
