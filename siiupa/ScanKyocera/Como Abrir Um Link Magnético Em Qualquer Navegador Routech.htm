Como abrir um link magnético em qualquer navegador

Internet / novembro 12, 2020 novembro 15, 2020
⌛ Reading Time: 4 minutes

Por muito tempo, os links magnéticos têm sido o esteio de todos os principais navegadores da web, tornando a vida um pouco mais fácil para as pessoas que baixam arquivos torrent. Mas o que exatamente são elos magnéticos? Como você os configura e abre? E como você altera o programa ao qual eles se vinculam? Revelamos tudo aqui.

Nota: Você precisará ter uma plataforma Bittorrent como Vuze, Utorrent ou Transmissão para usar links magnéticos.

O que é um link magnético?

Primeiro, uma rápida introdução. Links magnéticos e arquivos torrent andam de mãos dadas, mas não são exatamente a mesma coisa. Na verdade, se você usar um link magnético, nem precisará baixar um arquivo torrent para começar a baixar o conteúdo. Suas informações de identificação cruciais – o hash – são calculadas em um determinado site, e não no cliente de torrent, e enviadas diretamente para o programa Bittorrent.

Os benefícios disso são que um site não precisa mais hospedar o arquivo torrent original, basta fornecer o link. Para o usuário, isso elimina a necessidade de baixar um arquivo torrent antes de baixar o conteúdo e, para o site, economiza largura de banda porque não precisa hospedar arquivos em seus servidores.

A grande maioria dos sites de torrent atualmente usa links magnéticos.

Abrir links magnéticos no Chrome

Primeiro, você precisa se certificar de que o Chrome permitirá que você crie links magnéticos. Para fazer isso, vá para Configurações, clique no ícone do menu no canto superior esquerdo, “Avançado” e em “Privacidade e segurança”.

Em seguida, clique em “Configurações de conteúdo -> Manipuladores” e “Permitir que os sites peçam para se tornarem manipuladores padrão para protocolos (recomendado).”

Você só precisa encontrar um site de download de torrent que hospede links magnéticos (a maioria deles faz), selecionar o arquivo que deseja baixar e escolher “Download magnético”, se houver tal opção. Caso contrário, basta selecionar a opção de download padrão e há uma boa chance de que seja um link magnético.

Se for um link magnético que você está baixando, você receberá uma mensagem oferecendo a abertura do link magnético com a plataforma Bittorrent de sua escolha.

No nosso caso, é o Vuze. Quando o Chrome perguntar se você deseja abrir seu cliente Bittorrent, marque a caixa que diz “Sempre abra esses tipos de links no aplicativo associado”.

Abra os links magnéticos no Firefox

Para abrir links magnéticos no Firefox, você encontra novamente um arquivo torrent que deseja baixar e um pop-up deve aparecer perguntando com qual programa (geralmente o seu cliente de bit torrent) deseja abrir o arquivo. Para redefinir a associação e fazer com que o diálogo apareça novamente, faça o seguinte.

Na barra de endereço do Firefox, digite about:config, então, nessa página, pesquise por network.protocol-handler.expose.magnet. Clique com o botão direito nesta preferência e clique em “Alternar” para se certificar de que o “Valor” está definido como “falso”.

Abrir links magnéticos em outros navegadores

Se você já tem um cliente Bittorrent, o processo de associação de links magnéticos em outros navegadores é bastante semelhante. Basta encontrar o arquivo que deseja baixar, clicar nele e, se for um link magnético, seu cliente Torrent deverá abrir automaticamente ou você poderá receber uma mensagem perguntando se deseja abrir o arquivo magnético com seu aplicativo Bittorrent, como esta mensagem no Microsoft Edge ilustrado abaixo.

Se nada acontecer quando você clicar no link magnético ou receber a mensagem “este arquivo não tem um aplicativo associado a ele”, será necessário criar a associação do arquivo manualmente.

Vá para o Editor de registro do Windows (Ganhar + R, entrar regedit na caixa) e navegue até o seguinte local:

ComputerHKEY_CLASSES_ROOTMagnetshellopencommand

No painel de registro à direita em “Dados”, ele deve mostrar o diretório do seu aplicativo Bittorrent seguido por algum outro texto. Portanto, para Utorrent em seu diretório padrão, deve ser assim:

"C:UsersUserAppDataRoaminguTorrentuTorrent.exe" "%1" /SHELLASSOC

Se as informações em Dados não corresponderem ao diretório do cliente Bittorrent, clique com o botão direito em “(Padrão),” clique em “Modificar” e digite o diretório correto para o executável do cliente Bittorrent, certificando-se de que ele seja seguido por "%1" /SHELLASSOC.

Depois de fazer isso, todos os seus navegadores devem associar links magnéticos ao aplicativo Bittorrent de sua escolha.
